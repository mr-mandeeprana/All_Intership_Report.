## Projects Covered

### 1. BEUMER IO-Link Sensor Monitoring System (Digital Twin)
**Technologies:**  
Node-RED, Elasticsearch, Kibana, Docker, Kubernetes  

**Description:**  
A real-time industrial IoT monitoring system for IO-Link ultrasonic and vibration sensors. The system uses a Digital Twin model to collect, store, visualize, and analyze sensor data.

**Key Features:**
- Real-time monitoring (2-second updates)
- Historical trend analysis
- Threshold-based alerts
- Scalable containerized deployment

**Status:** ✅ Operational | ✅ Demo Ready  

---

### 2. Customer Feedback System (Digitalization)
**Technologies:**

**Description:**  
A full-stack web application for collecting and managing structured customer feedback across global BEUMER plants.

**Key Contributions:**
- Backend error resolution
- API optimization & validation
- MongoDB integration verification
- Cloud deployment on free platform

**Status:** ✅ Fully Deployed
